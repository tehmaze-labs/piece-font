We ♥ fonts

This is the font collection for the piece, a text-to-image render engine written in C for speed and for the lulz.

We've been scouring the internet for text mode fonts on old (8-bit) computers, software and other ROMs for text mode fonts. A great source has been the Multi Emulator Super System, and I would also like to thank Kyzrati from REXPaint for publishing his fonts.

Royalty free!

Well, maybe, it depends on your country. In a lot of countries the typeface is not copyrightable, but this varies per legal juresdiction. For more information, please refer to the comp.fonts FAQ.

We want your soul^wfonts!

If you think you have useful additions to the project, please get in touch with maze via IRC (EFNet, IRCNet, FreeNode, SmurfNet) or using maze@maze.io (PGP).

FAQ

Font $foo is not included, when will it be added?
Get in touch and provide us with enough details to be able to include your favourite font.
What do you use to convert the ROMs to YAMLs? What about the palettes?
Most of the tools are written in Python and make heavy use of the Pillow library.
Can you also provide the fonts in $my_favourite_format?
I'm sure we can, get in touch and we'll see what can be done.
Is it free?
Well, maybe, it depends on your country. In a lot of countries the typeface is not copyrightable, but this varies per juresdiction. For more information, please refer to the comp.fonts FAQ.
Can I include the fonts in my project?
Yes!, but please link to the project or at least mention in your documentation where you got the fonts from.
